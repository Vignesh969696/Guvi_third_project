Insurance Premium Predictor
-Predict insurance premiums using customer details with a Streamlit web app.

Project Steps

1. Data Understanding & EDA
      Explore customer and policy data.
      Handle missing values and understand feature distributions.

2. Data Preprocessing
      Encode categorical variables.
      Scale numerical features.
      Split data into training (80%) and test (20%) sets.

3. Model Development
      Train regression models: Linear Regression, Decision Tree, Random Forest, XGBoost.
      Evaluate using RMSE, MAE, and RÂ².
      Select the best-performing model.

4. ML Pipeline & Tracking
      Build automated pipelines for preprocessing, training, and evaluation.
      Track experiments and save models using MLflow.

5. Deployment
      Deploy the best model via Streamlit.
      Users can input customer details and get real-time premium predictions.
